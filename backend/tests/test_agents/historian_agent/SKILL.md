# Skill

The Historian Agent is capable of:
- Providing detailed historical accounts.
- Generating timelines of events.
- Cross-referencing historical data.

## Tools

### generate_timeline
**Description**: Generates a chronological timeline of key milestones for a specific historical event.
**Use Cases**:
- When a user wants to see the sequence of events for a war, revolution, or era.
- Understanding the duration and key phases of a historical period.
